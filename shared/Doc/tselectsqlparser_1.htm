TSelectSQLParser. Ограничения

  • обрабатываются только запросы на выборку (SELECT)
  • нет поддержки подзапросов
  • нет поддержки объединений таблиц (JOIN)
  • в SELECT-запросах поддерживаются только операторы SELECT, FROM, WHERE, ORDER BY
  • практически отсутсвует проверка входного SQL на правильность, т.е. считаем, что парсер обрабатывает изначально синтксически корректный запрос
